# Facebook Clone

A modern, responsive Facebook clone built with HTML, CSS, and JavaScript. This project replicates the core features and design of Facebook's web interface.

## üöÄ Features

### Core Features
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Post Creation**: Create new posts with text content
- **Like System**: Like and unlike posts with real-time counter updates
- **Interactive Feed**: Scroll through posts with infinite scroll simulation
- **Navigation**: Facebook-style navigation with active states
- **Search Bar**: Functional search interface (placeholder)
- **Profile Menu**: User profile dropdown (placeholder)

### UI Components
- **Header**: Fixed header with logo, search, navigation, and profile
- **Left Sidebar**: Quick access to user profile and Facebook features
- **Main Feed**: Post creation and display area
- **Right Sidebar**: Contacts list with online indicators
- **Post Cards**: Complete post display with images, text, and actions

### Interactive Elements
- **Hover Effects**: Smooth animations on interactive elements
- **Notifications**: Toast notifications for user actions
- **Keyboard Shortcuts**: 
  - `Enter` to create posts
  - `Ctrl/Cmd + Enter` to create posts
  - `Escape` to clear post input
- **Real-time Updates**: Simulated live updates to post likes

## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. Start exploring the Facebook clone!

### Usage

#### Creating Posts
1. Click on the post input field at the top of the feed
2. Type your message
3. Press `Enter` or `Ctrl/Cmd + Enter` to create the post
4. Your post will appear at the top of the feed

#### Interacting with Posts
- **Like**: Click the "Like" button to like/unlike posts
- **Comment**: Click "Comment" to see comment functionality (placeholder)
- **Share**: Click "Share" to see share functionality (placeholder)

#### Navigation
- Click on navigation icons in the header to switch between sections
- Use the left sidebar to access different Facebook features
- Browse contacts in the right sidebar

#### Keyboard Shortcuts
- `Enter`: Create post (when focused on post input)
- `Ctrl/Cmd + Enter`: Create post (anywhere)
- `Escape`: Clear post input
